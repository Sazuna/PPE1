    function FlagManager(flagNames) {
        var self = this;
        self.flags = {};
        if (Array.isArray(flagNames)) {
            for (var i = 0; i < flagNames.length; i++) {
                self.flags[flagNames[i]] = 2 << i;
            }
        } else if (typeof flagNames === 'object') {
            var i = 0;
            for (var p in flagNames) {
                if (flagNames.hasOwnProperty(p)) {
                    self.flags[flagNames[p]] = 2 << i;
                    i++;
                }
            }
        }
        self.value = 0;
        self.subscribers = {};

        function getFlag(flag) {
            if (Array.isArray(flag)) {
                return flag.reduce(function (result, f) {
                    return result | getFlag(f);
                }, 0);
            }
            return self.flags[flag] || flag || 0;
        }

        self.get = function (flag) {
            flag = getFlag(flag);
            return (self.value & flag) === flag;
        };
        self.set = function (flag) {
            self.value |= (getFlag(flag));
            setTimeout(function () {
                Object.keys(self.subscribers)
                    .map(function (mask) {
                        if (self.get(parseInt(mask))) {
                            self.subscribers[mask].forEach(function (cb) {
                                cb();
                            });
                            delete self.subscribers[mask];
                        }
                    }.bind(self));
            }.bind(self), 0);
        };
        self.subscribe = function (mask, callBack) {
            mask = getFlag(mask);
            if (parseInt(mask) && typeof callBack === 'function') {
                if (self.get(mask)) return callBack();
                self.subscribers[mask] = (self.subscribers[mask] || []);
                self.subscribers[mask].push(callBack);
            }
        };
    }

        function apsAttachVideoTargeting({
            bids,
            vastRequestUrl,
            targetingParameter = 'cust_params',
            separator = '&',
            operator = '=',
        }) {
            var url = new URL(vastRequestUrl);
            var searchParams = new app.utils.URLSearchParams(url.search);
            var targeting = bids.filter(bid => bid.mediaType === 'video')
                .map(bid => Object.keys(bid.targeting)
                    .reduce((acc, key) => `${acc}${separator}${key}${operator}${bid.targeting[key]}`, ''))
                .join('');
            var existingTargeting = searchParams.get(targetingParameter) || '';
            searchParams.set(targetingParameter, existingTargeting + targeting);
            url.search = searchParams;
            return url.toString()
        }

<div class="article-body ">

                                
        
        
        
                            


<p class="article-body__text article-body__text--entry article-body__text-middle">
        El patio pol&iacute;tico de IU est&aacute; revuelto. La asamblea federal que la organizaci&oacute;n celebra este fin de semana en Madrid ha creado ese clima, que, a buen seguro, se enrarecer&aacute; conforme pasen los d&iacute;as y se acerque la jornada final en la que se decida la nueva direcci&oacute;n nacional.
</p>        
        
        
                                
        
        
                    <div class="article-body--seo-closed">
        
                            


<p class="article-body__text ">
        Respecto a la asamblea federal de IU, el lector debe prestar especial atenci&oacute;n, pues el batiburrillo que hay formado es de &oacute;rdago. Ya conocen que <strong>Gaspar Llamazares</strong> se presenta a la reelecci&oacute;n de coordinador general de la federaci&oacute;n de izquierdas y que en esta ocasi&oacute;n cuenta con el apoyo de la direcci&oacute;n andaluza y de su coordinador regional, <strong>Diego Valderas</strong>, que en la asamblea de hace tres a&ntilde;os se puso en contra del dirigente nacional y dio su apoyo al secretario general del PCE, <strong>Francisco Frutos</strong>. Por el contrario, el ahora sector cr&iacute;tico de IU de Andaluc&iacute;a, que hace tres a&ntilde;os respald&oacute; la opci&oacute;n de <strong>Llamazares</strong>, est&aacute; en su contra y amenaza con presentar una lista alternativa a la del coordinador tras el acercamiento que &eacute;ste ha hecho hacia el sector de <strong>Frutos</strong> y de <strong>Felipe Alcaraz</strong>. Esa lista cr&iacute;tica la podr&iacute;a encabezar, si al final se presenta, el diputado cordob&eacute;s de IU, <strong>Luis Carlos Rej&oacute;n</strong>.
</p>        
        
        
                                
        
        
        
                            


<p class="article-body__text ">
        De modo que <strong>Rej&oacute;n</strong>, y otros cr&iacute;ticos de IU, como el presidente de la Gerencia de Urbanismo y ex coordinador provincial de la federaci&oacute;n de izquierdas, <strong>Andr&eacute;s Oca&ntilde;a</strong>, se han situado enfrente del l&iacute;der nacional, que en su candidatura quiere llevar a la alcaldesa de C&oacute;rdoba, <strong>Rosa Aguilar</strong>, que si no es como <em>n&uacute;mero dos    </em> si aparecer&aacute; entre los cuatro primeros de la candidatura oficialista. Se da por conocido que <strong>Llamazares</strong> ha ofrecido a la alcaldesa el segundo puesto de su lista, posici&oacute;n por la que est&aacute; pugnando tambi&eacute;n <strong>Francisco Frutos</strong>. Para evitar m&aacute;s incertidumbre y crispaci&oacute;n en el ambiente preasambleario, la regidora municipal, que nunca ha dicho que acepta el ofrecimiento de su dirigente nacional, est&aacute; dispuesta a ceder la segunda plaza al secretario comunista. As&iacute; las cosas, <strong>Rosa Aguilar</strong> tendr&aacute; enfrente tambi&eacute;n a <strong>Rej&oacute;n</strong> y a <strong>Oca&ntilde;a</strong>, por citar dos ejemplos significativos.
</p>        
                    <div id="nlt_signup_botton" class="article-body__text nx-piano--offer nx-piano--offer--signup-button-body"></div>
                    
        
                                
        
        
        
                            


<p class="article-body__text ">
        <strong>Valderas</strong>, por su parte, ha calificado de <strong>"incoherente"</strong> la postura que ha tomado <strong>Rej&oacute;n</strong> porque, seg&uacute;n dijo ayer en Sevilla, &eacute;ste <strong>"lleva tres a&ntilde;os apoyando las pol&iacute;ticas y las decisiones"</strong> que el coordinador general ha llevado a cabo durante su mandato. El dirigente andaluz confiesa que no comprende esa forma de entender la coherencia, pues, seg&uacute;n dijo, el parlamentario cordob&eacute;s ha apoyado las tesis de <strong>Llamazares</strong> en los consejos pol&iacute;ticos federales de IU.
</p>        
        
        
                                
        
        
        
                            


<p class="article-body__text ">
        Los cr&iacute;ticos se siente traicionados por el coordinador general porque hace tres a&ntilde;os le dieron su apoyo y ahora &eacute;ste da cobertura al secretario general del PCE. Adem&aacute;s, no le perdonar&aacute;n la posici&oacute;n que adopt&oacute; respecto al m&eacute;todo de elecci&oacute;n de delegados a las asambleas provincial y regional, pues mientras los cr&iacute;ticos propon&iacute;an que el n&uacute;mero de &eacute;stos fuese proporcional al n&uacute;mero de votos conseguidos en las elecciones municipales, los oficialistas apostaron porque ese n&uacute;mero fuese en proporci&oacute;n al n&uacute;mero de adscritos que tiene la organizaci&oacute;n en sus respectivas asambleas locales. Ante el dilema, la presidencia federal se decidi&oacute; por la segunda opci&oacute;n, que, seg&uacute;n la corriente disidente, favorec&iacute;a a <strong>Enrique Centella</strong> en C&oacute;rdoba y a <strong>Valderas</strong> en Andaluc&iacute;a.
</p>        
        
        
                                
        
        
        
                            


<p class="article-body__text ">
        Otra de las razones que esgrime este sector para influir en la asamblea de Madrid radica en el hecho de que observa cierta p&eacute;rdida de pluralidad en IU en favor del PCE tras el acuerdo alcanzado entre <strong>Llamazares</strong> y <strong>Frutos</strong>. A este respecto, la corriente cr&iacute;tica denuncia tambi&eacute;n que desde hace a&ntilde;os no se incorpora al proyecto de IU ninguna nueva organizaci&oacute;n.
</p>        
        
        
                                
        
        
        
                            


<p class="article-body__text ">
        Dicen que al perro flaco todo se le vuelven pulgas, pero aqu&iacute; queda algo m&aacute;s. Por una parte, el tradicional rechazo de las organizaciones a personas brillantes que despegan por s&iacute; mismas y son m&aacute;s dif&iacute;ciles de controlar; por otro, el posible aislamiento en que puede quedarse <strong>Rosa Aguilar</strong> que, aunque sea una estrella en Madrid, necesita mucho a su gente de C&oacute;rdoba. Y, finalmente, los extra&ntilde;os compa&ntilde;eros que proporcionan los intereses personales en el mundo de la pol&iacute;tica, donde la gente da bandazos sin complejos.
</p>
