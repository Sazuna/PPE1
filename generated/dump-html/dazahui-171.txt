<p><img loading="lazy" class="alignnone size-medium wp-image-3389160" src="https://www.sinchew.com.my/wp-content/uploads/2021/10/WhatsApp-Image-2021-10-31-at-10.48.05-1-600x327.jpeg" alt="" width="600" height="327" srcset="https://www.sinchew.com.my/wp-content/uploads/2021/10/WhatsApp-Image-2021-10-31-at-10.48.05-1-600x327.jpeg 600w, https://www.sinchew.com.my/wp-content/uploads/2021/10/WhatsApp-Image-2021-10-31-at-10.48.05-1-768x419.jpeg 768w, https://www.sinchew.com.my/wp-content/uploads/2021/10/WhatsApp-Image-2021-10-31-at-10.48.05-1.jpeg 866w" sizes="(max-width: 600px) 100vw, 600px" /></p>
<p>第一次去<span class="article-content-tag-links"><a class="tagClick" onclick="tagClick(this);" tag="适耕庄" ref="www.sinchew.com.my/?p=4386771" href="/tag/%e9%80%82%e8%80%95%e5%ba%84/">适耕庄</a></span>是二十多年前的事，当时同学父亲是渔夫，在家安排了渔夫粥底火锅，什么海鲜都有。</p><div style='text-align:center;padding-bottom:1.5rem !important' class="ads-frame mobile-ads-frame "><p class="dfp ad_header ">ADVERTISEMENT</p><div id='HPR1_M'><script type='text/javascript'>if(screen.width < 768){googletag.cmd.push(function() { googletag.display('HPR1_M'); });}</script></div></div>
<p>这是我第一次吃潮州人的粥底火锅，满满的鱼虾蟹大杂烩火锅也令我这小山姑开了眼界和味蕾，原来火锅可以如此清甜，可以把整锅给喝到一滴汤都不剩，却又意犹未尽。</p>
<figure id="attachment_4386786" aria-describedby="caption-attachment-4386786" style="width: 1024px" class="wp-caption alignnone"><img loading="lazy" class="wp-image-4386786 size-full" src="https://www.sinchew.com.my/wp-content/uploads/2023/01/71e5a4a7e983bde4bc9a-e4b893e6a08fefbc9ae9a9ace68b89e5a6b9e587bae997b8e59ca8e89589e8b596e59083e98082e88095e5ba84e9b1bce4b8b8e6b5b7-5.jpg" alt="7/1大都会-专栏：马拉妹出闸/在蕉赖吃适耕庄鱼丸海鲜面" width="1024" height="768" srcset="https://www.sinchew.com.my/wp-content/uploads/2023/01/71e5a4a7e983bde4bc9a-e4b893e6a08fefbc9ae9a9ace68b89e5a6b9e587bae997b8e59ca8e89589e8b596e59083e98082e88095e5ba84e9b1bce4b8b8e6b5b7-5.jpg 1024w, https://www.sinchew.com.my/wp-content/uploads/2023/01/71e5a4a7e983bde4bc9a-e4b893e6a08fefbc9ae9a9ace68b89e5a6b9e587bae997b8e59ca8e89589e8b596e59083e98082e88095e5ba84e9b1bce4b8b8e6b5b7-5-600x450.jpg 600w, https://www.sinchew.com.my/wp-content/uploads/2023/01/71e5a4a7e983bde4bc9a-e4b893e6a08fefbc9ae9a9ace68b89e5a6b9e587bae997b8e59ca8e89589e8b596e59083e98082e88095e5ba84e9b1bce4b8b8e6b5b7-5-768x576.jpg 768w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption id="caption-attachment-4386786" class="wp-caption-text">四大碗公的海鲜粥和鱼线，午餐很豪气啊！</figcaption></figure>
<p>那时我想，潮州粥原来是这种“明明是汤泡饭但叫做粥的粥”，后来当然知道是自己肤浅了，去过潮州见识过潮州粥，花样如此之多，又涨了见识。潮州人到底是有多爱吃粥啊？而且又能把海鲜甜味都融合在米饭里？</p>
<p>我喜欢适耕庄的美食，也喜欢这种粥，所以每每在吉隆坡发现有写适耕庄海鲜的店，都试过不少。</p><div id='anymind-mid2'></div>
<figure id="attachment_4386785" aria-describedby="caption-attachment-4386785" style="width: 1024px" class="wp-caption alignnone"><img loading="lazy" class="wp-image-4386785 size-full" src="https://www.sinchew.com.my/wp-content/uploads/2023/01/71e5a4a7e983bde4bc9a-e4b893e6a08fefbc9ae9a9ace68b89e5a6b9e587bae997b8e59ca8e89589e8b596e59083e98082e88095e5ba84e9b1bce4b8b8e6b5b7-4.jpg" alt="7/1大都会-专栏：马拉妹出闸/在蕉赖吃适耕庄鱼丸海鲜面" width="1024" height="768" srcset="https://www.sinchew.com.my/wp-content/uploads/2023/01/71e5a4a7e983bde4bc9a-e4b893e6a08fefbc9ae9a9ace68b89e5a6b9e587bae997b8e59ca8e89589e8b596e59083e98082e88095e5ba84e9b1bce4b8b8e6b5b7-4.jpg 1024w, https://www.sinchew.com.my/wp-content/uploads/2023/01/71e5a4a7e983bde4bc9a-e4b893e6a08fefbc9ae9a9ace68b89e5a6b9e587bae997b8e59ca8e89589e8b596e59083e98082e88095e5ba84e9b1bce4b8b8e6b5b7-4-600x450.jpg 600w, https://www.sinchew.com.my/wp-content/uploads/2023/01/71e5a4a7e983bde4bc9a-e4b893e6a08fefbc9ae9a9ace68b89e5a6b9e587bae997b8e59ca8e89589e8b596e59083e98082e88095e5ba84e9b1bce4b8b8e6b5b7-4-768x576.jpg 768w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption id="caption-attachment-4386785" class="wp-caption-text">海鲜粥米粒分明，汤底清甜，鱼虾都非常新鲜。</figcaption></figure>
<p>位于蕉赖Taman Bukit Anggerik的“适耕庄<span class="article-content-tag-links"><a class="tagClick" onclick="tagClick(this);" tag="鱼丸" ref="www.sinchew.com.my/?p=4386771" href="/tag/%e9%b1%bc%e4%b8%b8/">鱼丸</a></span>海鲜面饭店”（Ivy Sekinchan Special Fish Ball），位置在较偏的旧店区，旁边都是些小型工厂。</p><div id="adasia-mb-mrec2"></div><div id='AnyMind_PC_RecII'></div>
<p>不过，好吃的东西倒不怕人找不到，午饭时间这里也坐满了人，还是以熟客为主，大家坐下来就懂要吃什么，连看餐牌都省，不过发现点咖哩粉吃的人也不少。</p><div class='read-more-wrapper'>
<figure id="attachment_4386774" aria-describedby="caption-attachment-4386774" style="width: 1024px" class="wp-caption alignnone"><img loading="lazy" class="wp-image-4386774 size-full" src="https://www.sinchew.com.my/wp-content/uploads/2023/01/71e5a4a7e983bde4bc9a-e4b893e6a08fefbc9ae9a9ace68b89e5a6b9e587bae997b8e59ca8e89589e8b596e59083e98082e88095e5ba84e9b1bce4b8b8e6b5b7.jpg" alt="7/1大都会-专栏：马拉妹出闸/在蕉赖吃适耕庄鱼丸海鲜面" width="1024" height="768" srcset="https://www.sinchew.com.my/wp-content/uploads/2023/01/71e5a4a7e983bde4bc9a-e4b893e6a08fefbc9ae9a9ace68b89e5a6b9e587bae997b8e59ca8e89589e8b596e59083e98082e88095e5ba84e9b1bce4b8b8e6b5b7.jpg 1024w, https://www.sinchew.com.my/wp-content/uploads/2023/01/71e5a4a7e983bde4bc9a-e4b893e6a08fefbc9ae9a9ace68b89e5a6b9e587bae997b8e59ca8e89589e8b596e59083e98082e88095e5ba84e9b1bce4b8b8e6b5b7-600x450.jpg 600w, https://www.sinchew.com.my/wp-content/uploads/2023/01/71e5a4a7e983bde4bc9a-e4b893e6a08fefbc9ae9a9ace68b89e5a6b9e587bae997b8e59ca8e89589e8b596e59083e98082e88095e5ba84e9b1bce4b8b8e6b5b7-768x576.jpg 768w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption id="caption-attachment-4386774" class="wp-caption-text">小店在胡姬岭的旧店区内，午间多为熟客生意。 ​</figcaption></figure>
<p>小店主打海鱼和鱼丸，猪肉粉、客家面、咖哩面皆有，10令吉不到一碗有交易；另一页菜单则是价钱稍高的海鲜粉和粥，20令吉出头的价钱，上来一大碗公能把小鸟胃撑到。</p><div id='anymind-mid3'></div>
<p>例牌点<span class="article-content-tag-links"><a class="tagClick" onclick="tagClick(this);" tag="海鲜粥" ref="www.sinchew.com.my/?p=4386771" href="/tag/%e6%b5%b7%e9%b2%9c%e7%b2%a5/">海鲜粥</a></span>，底料有虾和石斑鱼肉，汤底鲜甜无比，店家还贴心地拿来一罐胡椒粉，说这是砂拉越胡椒粉，果然跟桌面上另一罐的辣度和香气都差很大，洒上去就是加持了汤底甜度，店家有心了。</p><div id="adasia-mb-mrec3"></div>
<figure id="attachment_4386783" aria-describedby="caption-attachment-4386783" style="width: 768px" class="wp-caption alignnone"><img loading="lazy" class="wp-image-4386783 size-full" src="https://www.sinchew.com.my/wp-content/uploads/2023/01/71e5a4a7e983bde4bc9a-e4b893e6a08fefbc9ae9a9ace68b89e5a6b9e587bae997b8e59ca8e89589e8b596e59083e98082e88095e5ba84e9b1bce4b8b8e6b5b7-3.jpg" alt="7/1大都会-专栏：马拉妹出闸/在蕉赖吃适耕庄鱼丸海鲜面" width="768" height="1024" srcset="https://www.sinchew.com.my/wp-content/uploads/2023/01/71e5a4a7e983bde4bc9a-e4b893e6a08fefbc9ae9a9ace68b89e5a6b9e587bae997b8e59ca8e89589e8b596e59083e98082e88095e5ba84e9b1bce4b8b8e6b5b7-3.jpg 768w, https://www.sinchew.com.my/wp-content/uploads/2023/01/71e5a4a7e983bde4bc9a-e4b893e6a08fefbc9ae9a9ace68b89e5a6b9e587bae997b8e59ca8e89589e8b596e59083e98082e88095e5ba84e9b1bce4b8b8e6b5b7-3-450x600.jpg 450w" sizes="(max-width: 768px) 100vw, 768px" /><figcaption id="caption-attachment-4386783" class="wp-caption-text">芫荽控必须加菜才够味的。</figcaption></figure>
<p>同事们吃海鲜米线，大口大口的嗦粉声音让我甚是怀念，吃米线我还是喜欢螺丝粉，都是各人的喜好；有加点炸鱼饼和大鱼丸，后者加了辣椒葱调味，口感弹爽鲜，小贵但挺好吃的。</p>
<p>最近吃东西，同事都会笑问我说这是米其林标准吗？每次都会翻白眼，喜欢吃的就继续去支持就好，好吃的东西永远会有人找出来，无论是米其林食探还是你自己。</p>
<p>而且，我不爱给美食按排名，这一家跟我合眼缘也很对胃口，那就足够了！</p>

    function initCarousel(elem){
        if ($('body').outerWidth() < 768) {
            var slider = $(elem).owlCarousel({
                margin: 10,
                autoWidth: true,
                mouseDrag: false,
                items: 1,
                dots: true,
                nav: true,
                navText: ["<div class='carousel-nav prev'></div>", "<div class='carousel-nav next'></div>"],
                responsive: {
                    0: {
                        nav: false
                    },
                    768: {
                        nav: true
                    }
                }
            }).trigger('refresh.owl.carousel');
            slider.on( 'drag.owl.carousel', function( event ) {
                document.ontouchmove = function( e ) {
                    e.preventDefault();
                }
            } );
            slider.on( 'dragged.owl.carousel', function( event ) {
                document.ontouchmove = function( e ) {
                    return true;
                }
            } );
            slider.on('translatecoordinate.owl.carousel', function(event){
                let target_stage = $(event.target).find(".owl-stage").first();
                let stage_width = target_stage.width();
                let stage_parent_width = target_stage.parent().width();
                if(stage_width <= stage_parent_width) return event.coordinate;
                let max_translateX = -1 * (stage_width - stage_parent_width);
                var matrix = event.coordinate;
                if(matrix < max_translateX ){
                    event.coordinate = max_translateX;
                }
                return event.coordinate;
            });
        }
    }

                }
