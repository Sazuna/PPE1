<h2><strong>中共“开闸门”放人出国 民忧“毒株大杂烩要来了”</strong></h2>
<p>自由亚洲电台报导，中国卫健委26日宣布将从明年1月8日起放宽中国的出入境后，中国各大“去哪儿”、“同程”、“携程”等旅行平台上，国际机票、签证等出境旅游相关的搜索量突然暴涨7至10倍不等，尤其是明年虎年黄历新年期间数据达到3年来高峰，这些热搜目的地包括日本、韩国、泰国，而主要搜索用户来自于<a href=https://www.secretchina.com/news/gb/tag/中国疫情 target='_blank'>中国疫情</a>“重灾区”北上广。
</p>
<p>彭博社也于12月28日以&lt;北京打开闸门，中国旅游需求跃升&gt;为题报导，根据“携程”（Trip.com）数据显示，27日上午从中国<a href=https://www.secretchina.com/news/gb/tag/出境 target='_blank'>出境</a>航班机票订单，比26日同时大增254%。前5大目的地中，赴新加坡机票订单大增600%，韩国、香港、日本和泰国增幅约400%。
</p>
<p>中国《证券时报》引述携程研究院人士说法称，26日晚上政府宣布恢复中国公民出境<a href=https://www.secretchina.com/news/gb/tag/旅游 target='_blank'>旅游</a>等措施后半小时内，前述5大地点和澳洲、英国等地，迅速成为搜寻量增加较快的热门地点。
</p>
<p>美国之音也报导，中共官方公布消息后中国网络社交平台微博上，不少人欢呼3年的出入境封锁终于结束，并留言写道“好久好久的恶梦，终于醒了”、“一个荒唐的时代终于落幕了”。但也有人担心打开国门后将造成疫情进一步恶化，“毒株大杂烩要来了”。
</p>

	    var c = ca[i];
	    while (c.charAt(0) == ' ') {
	      c = c.substring(1);

	      return c.substring(name.length, c.length);
	    }
	  }
	  return "";
	}
