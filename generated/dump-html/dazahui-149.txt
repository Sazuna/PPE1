function getScrollTop(){  
	var scrollTop=0;
	if(document.documentElement&&document.documentElement.scrollTop){  
		scrollTop=document.documentElement.scrollTop;
	}else if(document.body){  
		scrollTop=document.body.scrollTop;  
	}  
	return scrollTop;
}

function getClientHeight(){  
	var clientHeight=0;  
	if(document.body.clientHeight&&document.documentElement.clientHeight){  
		var clientHeight = (document.body.clientHeight<document.documentElement.clientHeight)?document.body.clientHeight:document.documentElement.clientHeight;
	}else  {  
		var clientHeight = (document.body.clientHeight>document.documentElement.clientHeight)?document.body.clientHeight:document.documentElement.clientHeight;      
	}  

	return clientHeight;  
}  

function getScrollHeight(){  
	return Math.max(document.body.scrollHeight,document.documentElement.scrollHeight);  
}

<p>　　当然，从某种角度来看，高希希也并没有说错。《露水红颜》里，他热衷于将这部电影打造成一个混搭了各种元素的大杂烩：爱情片、商战片、赛车片、悬疑片，然后再用一个韩剧式的故事结构包装起来。他将这部电影做成了一份麻辣烫，里面搭配了最受欢迎的涮菜，却没注意食材之间的相宜相克，导致肠胃敏感的食客上吐下泻。</p>
<p><strong>　　【剧情梗概】</strong></p>
<p>　　这部电影是根据张小娴的同名小说改编，这本爱情小说诞生在将近十年以前。在当时或许能算是新颖但十年之间，相同题材的故事实在太多。不过高希希说了，电影与小说完全是两件事情，也就是说，高希希觉得自己完全打破了原小说，二次创作。</p>
<p>　　而结果如何呢？事实上是，相信在很多人的成长经历中，一定看过至少一部与《露水红颜》故事概念相似的电影/电视剧：其实并不怎么缺钱用的咖啡店老板娘（<a href="http://quan.sohu.com/taghome/%E5%88%98%E4%BA%A6%E8%8F%B2" target="_blank">刘亦菲</a>饰演），因为一千多万的酬劳，需要勾引一位不愿意接受家族产业，一心想画画的富二代（Rain饰演）。原计划是让他喜欢上她，最后她再因为他穷，也没才华狠狠甩掉他，以此让富二代受挫，以便回去继承家业。</p>
<p>　　毫无意外地，刘亦菲虽然圆满完成了任务，但还是喜欢上了Rain，然后毫无意外地，Rain也知道了真相&hellip;&hellip;为了避免剧透，接下来的故事我们就不再一一赘述，因为故事说到这里，相信很多观众都可以凭过往的看片经验继续联想下去。（或许当你们带着自己的猜测走进电影院后，会意外发现，自己不去当编剧真的太可惜了。）</p>
<p><strong>　　【论剧情：这是一部科幻片！】</strong></p>
<p><span style="font-family: 幼圆;">　　男女主角自带人肉GPS导航，逻辑去哪儿了？</span></p>
<p>　　爱情电影里，男女主角的街角重逢是一件多么浪漫且有戏剧感的事情。巧遇一次是缘分，两次是注定，如果三次就有点跳戏了。但如果还有四次、五次，甚至在一些莫名其妙的场合都能重逢，那真的就是科幻片了。《露水红颜》自然属于后者。自从电影的开头，Rain开的车差点撞到小学同学刘亦菲这第一次“重逢”之后，两人之后的见面就朝着不可思议的方向发展下去。在上海，刘亦菲想要跳楼，恰好碰到了来找回忆的Rain；在外滩，Rain坐在赛车里，刚往前开没多久，刘亦菲突然冲出来，拦在了Rain的面前――刘亦菲竟然能在两辆长得一模一样且高速行驶的赛车面前，精准判断出哪一辆是Rain开的，不得不佩服她的好眼力。</p>
<p>　　最具有科幻气质的“重逢”莫过于快到结尾的地方，Rain千里迢迢跑去找刘亦菲吵架。争吵后的两人似乎有些意犹未尽，于是原本已经开车离开的Rain掉头回去。与此同时，刘亦菲也一把冲出房门，准备去追车&hellip;&hellip;从背后的风景判断，两人走的是不同的路，并且一前一后时间隔了有一会。但最终在某一个拐角，带着伤跑了可能不止一千米的刘亦菲竟然再次与坐在车里的Rain重逢！这样都能撞见，不得不怀疑这两人是否都在身体里装了人肉GPS导航系统。</p>
<p><span style="font-family: 幼圆;">　　被急刹车撞出抛物线，地心引力去哪儿了？</span></p>
<p>　　电影里有一处美轮美奂的撞车场景。一位美丽的女子撞上了一辆小轿车，然后慢镜头叙事开始，只见这位女子先是被撞出一条抛物线，然后从马路的左侧在空中划出一条美丽的曲线，然后掉入了右测的河里。这时候问题就来了：明明这辆小轿车已经采取了紧急制动模式，并且这不是高速公路，在一条有限速的普通公路上，一个体重看上去明显超过50KG的女孩，是如何能够被撞出一条那么高的抛物线呢？并且车是竖着开的怎么就撞成横着的抛物线了呢？是这位女子脚上安装了喷气装置，还是维也纳这座城市没有地心引力？</p>
<p><span style="font-family: 幼圆;">　　外滩可以赛黑车，警察去哪儿了？</span></p>
<p>　　为了增加电影的商业元素，电影里将穷得只剩下画的Rain设定成一个靠赛车赚钱的人。于是奇妙的一幕开始了，在上海的外滩，竟然有一条没有人烟的高速公路能够赛黑车，并且大晚上的所有热爱这一项目的人都集中在了这里。</p>
<p>　　赌博、赛车的超速与喧嚣、群众的呼喊，还有乱七八糟的护栏和垃圾，堂堂大上海怎么允许这样的事情发生？</p>
<p><strong>　　【聊台词：这是一部哲理片！】</strong></p>
<p>　　说完神展开的剧情之后，还有一些台词听过之后让人对自己的智商产生了深深的怀疑，具体心里活动如下：这句话说出来不是等于没说吗？难道有潜台词？可是&hellip;&hellip;想了很久还是觉得没有潜台词啊&hellip;&hellip;</p>
<p><span style="font-family: 幼圆;">　　“因为你是个画家”</span></p>
<p>　　说这句话的是刘亦菲饰演的角色：咖啡店老板娘说出来的。有一天刘亦菲实在看不下去Rain又穷又爱闹事，所以让他留下来在咖啡店工作。Rain在一杯咖啡的奶泡上用巧克力画了一个简笔涂鸦，刘亦菲大加赞赏地说“知道你为什么做得这么好吗？”当你以为她会说出一句让你眼前一亮的哲理时，她说：“因为你是个画家。”――这难道这不是整部电影在一开头就不停在告诉观众的事实吗？最可怕的是Rain竟然还若有所思地点了点头。</p>
<p><span style="font-family: 幼圆;">　　“老公会用一辈子的时间弥补缺憾”</span></p>
<p>　　刘亦菲和Rain一起去吃法餐，服务员误将藏着戒指的蛋糕端上了他们的桌子。刘亦菲很熟练地一秒钟就找到了藏在偌大蛋糕里的小钻戒，并把它藏了起来，对Rain说“订婚的时候没有这枚戒指，以后老公会用一辈子的时间来弥补这次的缺憾。”乍一听特别浪漫对不对？可是事实上如果没有这枚戒指，当女孩吃完整个蛋糕都没吐出戒指的时候，男孩的第一反应该是“糟糕这枚戒指不会被女孩吞下去了吧”，然后男孩告诉女孩真相，两人接下来两人应该直接去医院了。</p>
<p>　　电影里特地加了这样一个完全经不起推敲，只是为了矫情而矫情的桥段，往积极正面的方向来说，大抵是编剧在自黑，以此来讽刺那些因为矫情而忽略了智商的“清新癌”们。</p>
<p><span style="font-family: 幼圆;">　　“你的眼睛像孩子一样纯洁清澈，但你的心却难逃世俗”</span></p>
<p>　　说这句话的时候，是Rain在酒店里发现了刘亦菲与别人开房，心碎的时候说出来的一句分手台词。Rain在电影里，一直都是一个阳光大男孩的形象出现，全程只负责做一些很蠢却很浪漫的事情讨刘亦菲欢心，在分手时，竟然能说出这样一句文邹邹的话语也是太让人大跌眼镜。不仅如此，除了部分较为简单且要给脸部大特写的镜头是用中文对的口型之外，Rain全程都在说韩语，所以最直观的视觉体验是，Rain对着镜头胡说一气，然后一个陌生的声音飘出来，念了这样一句文艺到不行的话语，如何叫人不跳戏。</p>
<p><strong>　　【气氛：这是一部电视剧！】</strong></p>
<p><span style="font-family: 幼圆;">　　用音乐铺垫所有气氛</span></p>
<p>　　因为电影又有爱情又有商战又有悬疑成分，如何将这些成分恰如其分地糅合在一起，且能在短短一个半小时之内让观众马上转换情绪呢？导演想到了一个妙招，就是用不同氛围的音乐来进行铺垫，浪漫就用小提琴，抒情就用钢琴，煽情就用大提琴，紧张就用鼓点，并且这些音乐，统统要在画面开始的前一秒钟响起。这样一个懒办法，也是常常能在电视剧中看到的。《露水红颜》在表现悬疑桥段时，画面还未切换，凝重的音乐已经响起，虽然画面中出现的是再正常不过的演员开门入画，然后径直穿过房间的场面。那一刻，把耳朵捂上，光看画面，你是完全看不出任何变化的。</p>
<p><span style="font-family: 幼圆;">　　镜头语言莫名其妙，导演对演员吃相没信心？</span></p>
<p>　　另外顺嘴说一说高希希贫乏的镜头语言，这部电影几乎看不到电影感的最大症结就是镜头语言太贫乏，也没什么特别的炫技之处，有的地方还特别莫名其妙。在刘亦菲与Rain浪漫地吃着烛光晚餐分享彼此身世的时候，高希希拼命用旋转的移动旋转镜头拍出了一种紧张感。问题来了，这样的镜头语言常常是为了烘托紧张感、史诗感，可明明两个人是在那里掏心掏肺地、安静地说话和吃饭，这样摇来摇去只会让人产生一种联想――导演对两位演员的吃相没有信心。</p>
<p><strong>　　结语：高希希是灾难片界的良心</strong></p>
<p>　　《露水红颜》给人提供了一种这样的观影感受：抱着看爱情片的情绪进场，却怀着看完灾难片的心情离场。就如同我们观看《狂蟒之灾》后会怕蛇，看《不惧风暴》后会怕龙卷风，看《2012》会怕世界末日一样，看完这部电影后，最怕听到和看到的，莫过于导演“高希希”这三个字。</p>
<p>　　听说当初<a href="http://quan.sohu.com/taghome/%E8%B5%B5%E5%AE%9D%E5%88%9A" target="_blank">赵宝刚</a>银幕处女作《触不可及》在京举行过一场明星首映，许多与他合作过的演员纷纷在观后的饭局上，把导演、表演、故事等等全部赞了一遍。想必在今天上映之后，也一定会有这样的观影场和讨论局出现。对此，我只能说：看完《露水红颜》后还敢点赞的人，我敬你们是条汉子！</p>
